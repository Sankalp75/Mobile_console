# Icons Directory

Place app icons here for PWA support.

Recommended sizes:
- 192x192 (Android)
- 512x512 (Android)
- 180x180 (iOS)
- favicon.ico (16x16, 32x32)

Generate icons from a square PNG using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
